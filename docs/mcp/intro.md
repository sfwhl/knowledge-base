# MCP 协议详解

## 什么是 MCP

MCP (Model Context Protocol) 是一个开放协议，用于标准化 AI 模型与外部数据源、工具和服务的连接。

## 核心组件

- **Resources** - 数据源
- **Tools** - 工具
- **Prompts** - 提示词模板

## 架构

```
AI Model ↔ MCP Server ↔ Data Sources
```
